# Vue3-Vite-TypeScript-Element-Plus 项目初始化模板

这是基于 **Vue3 + Vite + TypeScript + Element-Plus** 搭建的企业级项目初始化模板，整合了完整的工程化配置、基础功能封装及规范约束，可直接用于中后台管理系统、业务平台的快速开发。

## 一、工程化核心配置模块
### 1. 路径与自动工具配置
- **路径别名配置**：通过`vite.config.ts`和`tsconfig.json`配置`@`映射`src`目录，简化文件引入路径。
- **安装自动导入**：借助`unplugin-auto-import`自动导入Vue/Element-Plus/Pinia等API，无需手动引入。

### 2. 组件与样式整合配置
- **整合Element Plus**：通过`unplugin-vue-components`自动按需引入组件与样式，无需手动注册。
- **整合SVG图标**：使用`vite-plugin-svg-icons`实现SVG图标组件化，支持按目录分类与动态渲染。
- **整合SCSS**：配置全局SCSS变量，无需手动导入即可在组件中使用，支持嵌套语法。
- **整合UnoCSS**：集成原子化CSS工具，通过类名快速编写样式，减少CSS代码量。

### 3. 状态与网络配置
- **整合Pinia**：配置Pinia状态管理，支持模块化Store与持久化存储。
- **环境变量**：通过`.env.*`文件区分环境，定义全局变量并通过`import.meta.env`访问。
- **跨域处理**：在`vite.config.ts`中配置代理，解决开发环境接口跨域问题。
- **整合Axios**：封装Axios实例，统一请求/响应拦截、错误处理，实现接口类型安全。
- **动态路由**：基于Vue Router实现动态路由，结合权限接口实现路由权限控制。


## 三、组件封装模块
- **wangEditor富文本**：封装为全局组件，支持自定义工具栏、内容回显与图片上传。
- **Echarts图表**：封装基础组件，支持配置项渲染、容器大小自适应与实例销毁。
- **图标选择器**：整合SVG图标，支持搜索与选择，适用于表单图标选择场景。

## 四、代码规范工具配置
- **Prettier**：专注代码格式化，统一缩进、换行、引号等格式细节，支持多文件类型，与 ESLint 配合，实现保存自动格式化，提升代码可读性。
- **Stylelint**：校验 CSS/SCSS 样式代码，检测语法错误、强制样式规范（如属性顺序、伪类合法性），支持 Vue 文件样式块校验，自动修复样式问题，保证样式代码规范与可维护性。
